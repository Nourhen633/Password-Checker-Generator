# Password Strength Checker & Generator 🔐

A Python project that allows users to:
- Check the strength of their own password  
- Generate a secure random password  
- Detect weak patterns like repeated characters or sequences (aaa, 123)  
- Ensure passwords contain uppercase letters, symbols, and digits  

This project is beginner‑friendly and demonstrates:
- Python OOP (classes, static methods)
- Input validation
- Random password generation
- Basic security logic
- Menu‑based CLI interaction

---

## 🚀 Features

### ✔ Password Strength Checker
Evaluates:
- Length  
- Repetition patterns  
- Sequential digits  
- Uppercase letters  
- Symbols  

Returns:
- **Weak**
- **Medium**
- **Strong**

### ✔ Password Generator
- Generates passwords between 8–16 characters  
- Uses letters, digits, and symbols  
- Ensures the generated password is **not weak**
